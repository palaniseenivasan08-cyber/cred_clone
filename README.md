# CRED Website Clone

A pixel-perfect, fully responsive React clone of the CRED fintech website with premium animations and NeoPOP design system.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🛠 Tech Stack

- **React 18** – UI framework
- **Vite** – Fast build tool
- **Tailwind CSS** – Utility-first styling
- **Framer Motion** – Premium animations
- **react-intersection-observer** – Scroll-triggered animations

## 📁 Project Structure

```
src/
├── components/
│   ├── Navbar.jsx            # Sticky nav with scroll effects
│   ├── HeroSection.jsx       # Fullscreen hero with floating cards
│   ├── AboutSection.jsx      # Trust & member profile section
│   ├── FeaturesSection.jsx   # Interactive feature showcase
│   ├── StatsSection.jsx      # Animated counters & marquee
│   ├── NeoPOPSection.jsx     # NeoPOP design showcase
│   ├── TestimonialsSection.jsx # Auto-rotating testimonials
│   ├── SecuritySection.jsx   # Security terminal & cards
│   ├── DownloadSection.jsx   # CTA & eligibility check
│   ├── Footer.jsx            # Full footer with links
│   └── CustomCursor.jsx      # Gold custom cursor
├── App.jsx
├── main.jsx
└── index.css                 # Global styles + animations
```

## ✨ Key Features

- **Dark premium theme** with gold (#D4AF37) accent
- **NeoPOP design system** – 3D shadows, glassmorphism
- **Framer Motion animations** – page load, scroll reveal, hover
- **Custom gold cursor** (desktop only)
- **Interactive feature tabs** with live previews
- **Animated stat counters**
- **Auto-rotating testimonials**
- **Security terminal** animation
- **Floating credit cards** in hero
- **Fully responsive** – mobile, tablet, desktop
- **Smooth scroll** & parallax effects
